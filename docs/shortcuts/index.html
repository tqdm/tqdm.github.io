trange

trange(*args, **kwargs)

A shortcut for tqdm(xrange(args), *kwargs). On Python3+ range is used instead of xrange.

tgrange

tgrange(*args, **kwargs)

A shortcut for tqdm.gui.tqdm(xrange(*args), **kwargs). On Python3+, range is used instead of xrange.

tnrange(*args, **kwargs)
