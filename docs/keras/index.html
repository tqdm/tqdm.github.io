TqdmCallback

TqdmCallback(self,
             epochs=None,
             data_size=None,
             batch_size=None,
             verbose=1,
             tqdm_class=tqdm)

keras callback for epoch and batch progress

__init__

TqdmCallback.__init__(epochs=None,
                      data_size=None,
                      batch_size=None,
                      verbose=1,
                      tqdm_class=tqdm)

Parameters

  • epochs : int, optional
  • data_size : int, optional
    Number of training pairs.
  • batch_size : int, optional
    Number of training pairs per batch.
  • verbose : int
    0: epoch, 1: batch (transient), 2: batch. [default: 1]. Will be set to 0 unless both data_size and batch_size are given. tqdm_class : optional tqdm class to use for bars [default: tqdm.auto.tqdm].
